# Parkinson's Disease Detection using Machine Learning

## Problem
Parkinson’s disease affects voice patterns.  
This project predicts whether a patient has Parkinson’s using biomedical voice measurements.

## Dataset
195 patient samples.

Target:
0 → Healthy  
1 → Parkinson's Disease

## Models Implemented
Decision Tree  
Logistic Regression  
Random Forest

## Methodology
1. Data preprocessing
2. Feature scaling
3. Train-test split
4. Model training
5. Model comparison

## Evaluation Metric
Accuracy

## Result
The models were compared and the best performing model was selected for prediction.
