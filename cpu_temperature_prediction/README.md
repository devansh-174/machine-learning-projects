# CPU Temperature Prediction using Machine Learning

## Problem
Server CPU overheating can lead to hardware failures.

This project predicts CPU temperature using system workload metrics.

## Features

CPU Usage  
Memory Usage  
Clock Speed  
Ambient Temperature  
Voltage  
Current Load  
Cache Miss Rate  
Power Consumption

## Models Implemented

Random Forest Regression  
Decision Tree Regression  
Linear Regression

## Evaluation Metrics

MAE  
RMSE  
R² Score

## Result

The best performing model was selected based on R² score.

## Technologies

Python  
Pandas  
Scikit-learn  
Matplotlib
