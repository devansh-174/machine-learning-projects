# Movie Recommendation System

## Problem
Streaming platforms recommend movies based on user preferences.

This project builds a recommendation system using movie genres and user ratings.

## Models Implemented

1. TF-IDF Genre Similarity
2. Sentence Transformer Embedding Similarity

## Methodology

1. Data preprocessing
2. Genre vectorization
3. User preference profile
4. Cosine similarity ranking

## Evaluation

Precision@10 metric was used to evaluate recommendation quality.

## Technologies

Pandas
