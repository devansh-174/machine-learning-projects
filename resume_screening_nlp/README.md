# AI Resume Screening System

## Problem

Recruiters receive thousands of resumes.

This project ranks candidates by comparing resumes with job descriptions using NLP.

## Pipeline

1. Text preprocessing
2. Skill extraction
3. SentenceTransformer embeddings
4. Cosine similarity matching

## Output

Similarity score  
Matched skills  
Missing skills  
Candidate ranking

## Technologies

Python  
Pandas  
Scikit-learn  
SentenceTransformers
